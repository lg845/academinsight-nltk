About AcademInsight - NLTK

AcademInsight NLTK (NLTK version) allows users to classify texts into disciplines or communicative purposes. The interface only works with raw texts. This means that the user has to copy and paste the text into the box provided. The interface, then, outputs an analysis of the text with the likelihood of the text being part of a given discipline or communicative purpose. The interface was created to be used by teachers and students, therefore, along with the results for the classification the interface provides examples of relevant grammatical features for the text classification into the discipline or purpose categories.

In contrast with the Biber Tagger interface, this classifier does not have a careful feature selection running in the background, and should be used with caution.
